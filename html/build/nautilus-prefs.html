File manager preferences

File manager views preferences — Specify the default view, sort order, and zoom levels for the file manager.File manager behavior preferences — Single-click to open files, run or view executable text files, and specify trash behavior.File manager display preferences — Control icon captions and the date format used in the file manager.File manager list columns preferences — Control what information is displayed in columns in list view.File manager preview preferences — Control when thumbnails and previews are used for files.Open an application when you plug in a device — Automatically run applications for CDs and DVDs, cameras, audio players, and other devices and media.
