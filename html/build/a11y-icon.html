Show the universal access menu

The universal access menu allows quick access to turn on or off various accessibility settings.

The universal access menu can be added to the top panel.

  1. Click the icon at the very right of the panel and select System Settings.

  2. Click Universal Access.

  3. Select the Typing tab.

  4. Select Turn on accessibility features from the keyboard.
