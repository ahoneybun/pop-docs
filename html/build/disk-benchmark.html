Test the performance of your hard disk

To test the performance of the hard disk, go to the dash and open the Disk Utility.

To run a benchmark in Disk Utility:

  1. Click on the hard disk in the Storage Devices list. Information will appear about the disk and its status under Drive.

  2. Choose Benchmark.

  3. Click Start Read-Only Benchmark or Start Read/Write Benchmark.

    The read-only benchmark will only test how fast data can be read off the disk. The read/write benchmark will also test how fast data can be put onto the disk, so it will take longer to complete.

When the test is finished, the results will appear on the graph. These results tell you how fast files were read or written, and how long it took. The time is measured in Milliseconds (ms) and data transfer speed is measured in Megabytes per Second (MB/s).

After the test is finished, the following statistics will be listed below the graph:

  • Minimum Read Rate in Megabytes per Second

  • Maximum Read Rate in Megabytes per Second

  • Average Read Rate in Megabytes per Second

  • Last Benchmark in Seconds

  • Minimum Write Rate in Megabytes per Second

  • Maximum Write Rate in Megabytes per Second

  • Average Write Rate in Megabytes per Second

  • Average Access Time in Milliseconds
