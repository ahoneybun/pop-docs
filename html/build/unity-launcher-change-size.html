Change the size of icons in the launcher

The size of the icons in the launcher cannot be changed by default, but you can install software that will allow you to change the size of the icons. To install the necessary software and change the size of the icons, complete these steps:

  1. Install the CompizConfig Settings Manager (ccsm) application from the Software Center.

    If you need assistance with installing the application, consult the software installation help, or review the Ubuntu Software Center Manual.

  2. After you have installed the program, open the Dash.

  3. Search for CompizConfig Settings Manager, then click its icon from the results to start it.

  4. In the Desktop section of the CompizConfig Settings Manager, click Ubuntu Unity Plugin, and select the Experimental tab.

  5. Move the Launcher icon size slider to increase or decrease the size of the Launcher icons.

    The default Launcher icon size is 48.

Your changes to the size of the icons will be applied immediately. Click the Back and Close buttons to exit the CompizConfig Settings Manager.
