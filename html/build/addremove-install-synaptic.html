Use Synaptic for more advanced software management

Synaptic Package Manager is more powerful and can do some software management tasks which Ubuntu Software Center cannot. Synaptic's interface is more complicated and does not support newer Software Center features like ratings and reviews and thus is not recommended for use by those new to Ubuntu.

  1. Open the Dash by pressing the meta key.

  2. Search for Synaptic from the search bar at the top of the Dash.

  3. When the Synaptic application appears in the list, click on it to open it. This will launch the application. If prompted to do so, enter your password.

  4. When Synaptic opens, click Search to search for an application, or click Sections and look through the categories to find one.

  5. Right-click the application that you want to install and select Mark for Installation.

  6. If you are asked if you would like to mark additional changes, click Mark.

  7. Select any other applications that you would like to install.

  8. Click Apply, and then click Apply in the window that appears. The applications that you chose will be downloaded and installed.

For more information about using Synaptic, consult the Synaptic Manual.
