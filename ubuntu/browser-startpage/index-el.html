Καλωσορίσατε στο Ubuntu 8.04 LTS!

Το έργο Ubuntu βασίζεται στις ιδέες που πηγάζουν από τη φιλοσοφία Ubuntu: ότι το λογισμικό πρέπει να είναι διαθέσιμο δωρεάν, ότι τα εργαλεία λογισμικού πρέπει να μπορούν να χρησιμοποιηθούν από τους ανθρώπους στην μητρική τους γλώσσα, και ότι οι άνθρωποι πρέπει να έχουν την ελευθερία να προσαρμόζουν και να αλλάζουν το λογισμικό τους με οποιοδήποτε τρόπο θελήσουν.

Πως θα βρείτε βοήθεια για το Ubuntu:

Υπάρχουν πολλοί τρόποι να βρείτε βοήθεια για το Ubuntu.

Αποκτήστε πρόσβαση στους οδηγούς και την τεκμηρίωση του Ubuntu επιλέγοντας «Βοήθεια» από το μενού «Σύστημα». Μπορείτε ακόμα να βρείτε βοήθεια στις ιστοσελίδες της επίσημης τεκμηρίωσης του Ubuntu και στην τεκμηρίωση που προσφέρει η κοινότητα.

Αν προτιμάτε να έρθετε σε επαφή με άλλους χρήστες, μπορείτε να χρησιμοποιήσετε τη λίστα ταχυδρομείου του ubuntu-gr (λίστες ηλεκτρονικού ταχυδρομείου στην αγγλική γλώσσα), φόρουμ ή να συνομιλήσετε με τα μέλη της κοινότητας στο κανάλι IRC #ubuntu-gr (για συνομιλία στην αγγλική γλώσσα #ubuntu), στο δίκτυο IRC Freenode.

Μπορείτε επίσης να βρείτε εμπορική υποστήριξη για το λογισμικό.

Συμμετάσχετε στο Ubuntu:

Η κοινότητα του Ubuntu αποτελείται από προγραμματιστές, συγγραφείς τεκμηρίωσης, μεταφραστές, καλλιτέχνες που παράγουν γραφικά, και κυρίως, από τους ανθρώπους που χρησιμοποιούν το Ubuntu στην καθημερινή τους ζωή. Σας προσκαλούμε να συμμετάσχετε σε αυτή την κοινότητα και να βοηθήσετε το Ubuntu να εξελιχθεί σε ένα λειτουργικό σύστημα που θα καλύπτει καλύτερα τις ανάγκες σας.

Για να μάθετε πως μπορείτε να συμμετάσχετε, επισκεφθείτε τη σελίδα συμμετοχής στην κοινότητα του Ubuntu!
